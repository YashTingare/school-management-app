# 🎓 iTeach — School Management System

A Python-based school management system with a CLI and Streamlit web UI.

("https://school-management-app-bfun5svfmwt8nqtutojunh.streamlit.app")

## 🚀 Run

```bash
pip install streamlit pandas
streamlit run app.py   # Web UI
python main.py         # CLI(Command Line Interface)
```

## ✨ Features
- Register students & teachers
- Add/update grades
- View profiles & all records
- Data saved to `school_data.json`

## 🛠️ Stack
Python · Streamlit · JSON
